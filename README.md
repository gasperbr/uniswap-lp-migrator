# Uni to Ekubo Migrator

Migrates Uniswap LP NFT positions to Ekubo on Ethereum:

- Uniswap V3 -> Ekubo
- Uniswap V4 -> Ekubo

Core contract: `src/UniToEkuboMigrator.sol`  
Interface: `src/interfaces/IUniToEkuboMigrator.sol`
